# Лендінг курсу подології

Готовий статичний лендінг (один файл `index.html`, без залежностей).

## Як завантажити на сервер
Просто скопіюйте `index.html` у корінь хостингу. Працює на будь-якому:
- звичайний хостинг / FTP — покладіть файл у `public_html`;
- GitHub Pages, Netlify, Vercel — перетягніть файл/папку;
- відкривається і локально подвійним кліком.

## Що замінити перед запуском (пошук по тексту `ВАШ_АКАУНТ` та `TODO`)
1. **Instagram** — `https://instagram.com/ВАШ_АКАУНТ` → ваш профіль (кілька місць).
2. **Telegram** — `https://t.me/ВАШ_АКАУНТ` → ваш акаунт або `https://t.me/+380XXXXXXXXX`.
3. **Блок «Викладач»** — замініть `[Ім'я...]`, опис і цифри в `t-stats`.
4. **Фото** — зараз стоять акуратні заглушки (блоки з підписами `Фото...`).
   Щоб поставити реальні фото — створіть папку `assets/` і замініть `<div class="ph">...</div>`
   на `<img src="assets/hero.jpg" alt="...">` (так само для викладача).
5. **OG-зображення** (для красивого прев'ю у соцмережах) — розкоментуйте `og:image` у `<head>`
   і покладіть `assets/og-cover.jpg` (рекоменд. 1200×630).

## Meta Ads / Pixel
- У `<head>` є закоментований блок **Meta Pixel** — вставте свій `PIXEL_ID` і розкоментуйте.
- Унизу в `<script>` є закоментований трекінг події **Lead** на кліки CTA — розкоментуйте після підключення Pixel.

## Структура екранів
Офер → Користь → Програма (День 1 онлайн / День 2 офлайн) → Викладач → Сертифікат →
Ціна та дата → FAQ → фінальний заклик. Липкий хедер + плаваюча панель CTA на мобільному.
